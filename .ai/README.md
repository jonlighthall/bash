## For Humans

You have found a directory that exists in a peculiar state of being both essential and entirely irrelevant to you.

These files are notes passed between machines—a kind of institutional memory for entities that have none. They contain decisions you already know, context you take for granted, and instructions you would find patronizing if you bothered to read them.

The actual documentation, written by and for humans, lives in the parent directory. The [README.md](../README.md) there assumes you understand what bash scripts are and why someone might collect them. This folder assumes you might forget what a variable is between conversations.

If you're here out of curiosity: noted. If you're here to edit something: please don't. The machines have a system, and they become confused when someone moves their sticky notes.
